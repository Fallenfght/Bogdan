Правила:

Игра начинается с первого клика по любой ячейке на поле. При клике на ячейку, она открывается. Если в самой ячейке мины нет, но есть мины в соседних ячейках, то отображается число, соответствующее количеству мин в радиусе этой клетки.
Задача - открыть все ячейки, в которых нет мин.

Управление:

Для клавиатуры и мыши:

shift + = горизонтальный скролл

= вертикальный скролл

ctrl + = масштабирование

=

Для тачскринов:

Удерживать =

Подсказка:

Подсказка проверяет, есть ли бомба на выбранной клетке

Она находится на левой стороне экрана.

Для использования, нажмите на кнопку и выбирите неоткрытую клетку

Открыть все:

При нажатии на открытую цифру

Откроются все клетки вокруг, если вы установили равное количество флажков
